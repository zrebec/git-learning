• Working directory
    Space on your local computer inside your project directory.
    git add
  • Staging area
    Space between your local dicretory and local repository (Your files are still not in your git repostiroty but git knows about that).
    git commit
  • Local repository
    Space in your local git repository (still is just in your computer your local disk but now git now commits changes and you can revert to previous version e.g.
    git push
  • Remote repository
    Now your files are finally in remote repository (like github, gitlab or bitbucket, etc...) Now your files and all changes are safe in cloud.
